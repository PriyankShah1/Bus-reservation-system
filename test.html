Here's a list of links and descriptions for some of the free tools I use.

Code Editors:

  • Atom.io - A free, open-source code editor for Windows and Mac.

  • Text Wrangler (now BBEdit) - The TextWrangler code editor for Mac operating systems is being retired, in favor of its predecessor, called BBEdit.

  • Brackets - Open Source Code Editor for Mac.

  • Notepad++ - An excellent code editor for Windows.

FTP Programs:

  • sFTP Client Lite - If you search for that under Chrome > Settings >
    More Tools > Extensions, you'll see that it's a Chrome Extension. They also have a standalone program: https://www.sftpclient.io/

  • Filezilla - Filezilla is a cross-platform, free, FTP program.

  • CyberDuck - Another free, popular FTP program.

Image Editing Software:

  • Pixlr Editor - An excellent image editor, based on Flash and used in any browser.

  • Fotor Image Editor - Fotor  is another browser-based image editor, no Flash required. Fotor also has standalone apps for Windows and Mac.
